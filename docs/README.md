# Clyth Documentation Site

This directory is reserved for the future GitHub Pages documentation site.

The README should remain a concise project introduction and release-facing overview. Longer material should move here as Clyth grows, including:

- language guide
- runtime module guide
- module/package authoring guide
- compiler architecture notes
- registry and package security documentation
- release notes

The 0.4.0 alpha release keeps a concise root-level `RELEASE_NOTES.md`. Longer release pages can move under this documentation site as the project grows.
