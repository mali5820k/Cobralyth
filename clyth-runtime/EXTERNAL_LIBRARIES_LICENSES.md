# Clyth Runtime External Library Licenses

This file reproduces the complete upstream license text for every third-party library currently vendored or named as an active runtime backend. It must be updated whenever a runtime dependency is added, removed, upgraded, or relicensed.

Clyth runtime modules prefer C, Clyth, and musl-compatible C ABI bindings. Official runtime modules avoid C++ runtime dependencies. Project names are used only for attribution and architecture documentation; Clyth is not affiliated with, endorsed by, or sponsored by these projects or their maintainers.
